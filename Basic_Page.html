Basic Structure

how to combine p5 with arduino

Download the latest p5.serialcontrol app
(pick top Beta link and use “Darwin” for Mac).
If it complains about it being a “dangerous” file
go to chrome://downloads/ and say it is okay.
Maybe save it to your applications.

Double click on the p5.serialcontrol app.
You might have to go into security system preferences
and to allow this to run.

It will relay things to P5 using a communication style
the Browser is allowed to use called WebSockets.
Within P5 you will use P5.js serialport library to
connect to this the Serial Control Server.
